BTCIndicator
============

Simple bitcoin price watcher for ubuntu


![screenshot](https://raw.github.com/ynh/BTCIndicator/master/screenshot.png)


### Run
````
$ python indicator.py &
````


### Supports

- mtgox

- bitstamp

- futures796

- btcchina

- okcoin

- chbtc

- fxbtc

- btctrade

- btc100
